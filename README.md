# Wallpaper_API_App

## Overview
This JavaFX application allows users to browse, view, and download high-quality car wallpapers. The app connects to two APIs(Kelly Blue Book & an image gallery) to 
fetch a wide range of images, ensuring users can easily find and set wallpapers.  

The project emphasizes efficient data fetching, user-friendly design, and maintainable code to support team collaboration and scalability. Development was done using 
UGA's Unix environment (Odin) with Emacs for editing and managing project documents.

> Note: The full source code repository is private due to UGA school policy.

## Features
- Connects to two APIs to retrieve high-resolution car images.
- Allows users to download images for personal use as wallpapers.
- Responsive and intuitive UI for browsing large image collections.
- Structured and documented code following coding standards to ensure maintainability and team collaboration.

## Skills Demonstrated
- JavaFX GUI development
- Cloud API integration for real-time image retrieval
- Efficient handling and caching of large datasets
